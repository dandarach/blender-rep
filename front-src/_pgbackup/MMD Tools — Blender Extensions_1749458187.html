AboutVersion History

MMD Tools contains various functions for editing MMD (MikuMikuDance) models.

  • Import/Export
    • Model data (.pmd, .pmx)
    • Motion data (.vmd)
    • Pose data (.vpd)
  • Chop and Join model parts
  • Rigid body editing
  • Material editing
  • Motion editing
