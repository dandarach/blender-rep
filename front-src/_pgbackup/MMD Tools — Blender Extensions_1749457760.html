MMD Tools

Utility tools for MMD model editing.
Add-on by MMD
About What's New Permissions Reviews Version History

MMD Tools contains various functions for editing MMD (MikuMikuDance) models.

  • Import/Export
    • Model data (.pmd, .pmx)
    • Motion data (.vmd)
    • Pose data (.vpd)
  • Chop and Join model parts
  • Rigid body editing
  • Material editing
  • Motion editing

What's New

4.3.7 June 8th, 2025
  • Improve Import/Export Workflow with User Options and VMD Bug Fixes in https://github.com/MMD-Blender/blender_mmd_tools/pull/222
  • Vertex Color Import/Export, Sharp Normals Fix, and Enhanced Frame Range, Bone Order, and PMX/VMD Export in https://github.com/MMD-Blender/blender_mmd_tools/pull/223
  • Add IK angle limit export options, fix camera driver cycles, align morph export order in https://github.com/MMD-Blender/blender_mmd_tools/pull/228
